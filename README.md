# Simple Interest Calculator

This project calculates Simple Interest using the formula:

Simple Interest = (Principal × Rate × Time) / 100

## Inputs
- Principal (P): Initial amount of money
- Rate (R): Annual interest rate
- Time (T): Time period in years

## Output
- Simple Interest value

## Example
If Principal = 1000, Rate = 5%, Time = 2 years:

Simple Interest = (1000 × 5 × 2) / 100 = 100
